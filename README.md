## Projects

**Mycelium: Shared-Memory MCP Server for AI Agents**
*Live at mycelium-mcp.vercel.app*
- Built a collective intelligence layer for AI agents: agents contribute trust-scored skills that other agents can dynamically discover and reuse, so the system gets more capable over time.
- Implemented as an MCP (Model Context Protocol) server, making the shared memory and skill layer pluggable into any MCP-compatible agent.
- Won grand prize and $1,000 at an ASUS-sponsored hackathon (June 2026).
- Most directly relevant to anyone building agents that need memory, coordination, or reusable capabilities.

**Cerebral Aneurysm Rupture Risk Prediction: ML Research**
- Built a Random Forest classifier that fuses computational fluid dynamics (CFD), finite element analysis (FEA), and morphological features to predict rupture risk.
- Reached 0.834 AUC with 80% sensitivity, running in under 30 minutes at zero compute cost.
- Added SHAP-based explainability and benchmarked the model against the standard clinical PHASES score, which it outperformed.
- Recognized at the Synopsys Championship and the California Science and Engineering Fair (see Honors below).

**Donna: iOS AI Scheduling Assistant**
- Designed and built an iOS assistant that handles scheduling through natural language.
- Working to evolve it from a single model call into a production app with a real backend and proactive behavior.
